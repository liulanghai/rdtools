# RdTools Chrome Extension

RdTools 是一款专为开发者打造的高效 Chrome 扩展程序。它集成了功能强大的 **JSON 编辑器**、**Base64 工具** 以及 **Authorization 生成器**，旨在通过极致的交互体验和数据精度保护，简化日常开发与调试流程。

## 核心功能

### 1. 深度优化的 JSON 编辑器
*   **高精度 BigInt 支持**：首创性地解决了 JavaScript `Number` 类型对长整型数字（超过 15 位）的精度丢失问题。无论是展示、编辑还是复制，都能确保长整数（如 ID、Snowflake 算法结果）原样保留。
*   **智能树形视图**：
    *   **原样展示**：树形视图中的字符串完整保留转义字符（如 `\n`、`\"`），真实还原数据状态。
    *   **嵌套 JSON 自动解析**：自动检测并递归渲染 Value 中的 JSON 字符串，使其像普通对象一样可折叠、可探索。
*   **交互式搜索**：
    *   **精准高亮**：支持对 Key 和 Value 进行字符级高亮展示。
    *   **路径追踪**：搜索命中时，自动展开所有层级的父节点，确保目标一目了然。
*   **双模式编辑**：
    *   **行内修改**：简单的数值、布尔值及短文本支持点击后直接修改。
    *   **弹窗编辑 (Modal)**：长文本或嵌套 JSON 字段自动触发悬挂式弹窗。弹窗内支持 **反序列化格式化 (Prettify)** 编辑，保存时自动同步回原始各字段。
*   **实时校验**：输入非法 JSON 时实时红框提醒，并显示详细的语法分析错误。
*   **静默反馈**：复制操作采用 1 秒自动消失的 Toast 提示，不干扰操作流。

### 2. Base64 编解码
*   **双向转换**：支持文本与 Base64 字符串的快速互转。
*   **Unicode 友好**：原生支持中文字符、表情符号等 Unicode 内容，杜绝乱码。

### 3. Authorization 生成器
*   **Basic Auth 快速生成**：输入用户名和密码，一键生成标准 `Basic` 认证字符串，直连接口测试。

## 浏览器集成体验

*   **侧边栏默认开启**：点击插件图标默认在 Chrome 侧边栏（Side Panel）打开，完美实现一边调试、一边查看文档的沉浸式体验。
*   **全屏响应式布局**：针对侧边栏和全屏标签页进行了专门的 UI 适配，采用上下布局最大化利用纵向空间。

## 安装指南

1.  下载本项目代码。
2.  在 Chrome 中访问 `chrome://extensions/`。
3.  开启右上角的 **"开发者模式"**。
4.  点击 **"加载已解压的扩展程序"**，选择本项目根目录。

## 如何启动
*   **侧边栏模式**：直接点击浏览器工具栏的 RdTools 图标。
*   **全屏模式**：在插件图标上右键，选择相应的功能菜单，或直接在侧边栏中操作。
